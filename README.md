# 🔬 Scientific Calculator

A powerful and user-friendly **Scientific Calculator** built to perform both basic and advanced mathematical operations. This project is designed for students, engineers, and anyone who needs quick and accurate calculations.

---

## 🚀 Features

* ➕ Basic Operations
  Addition, Subtraction, Multiplication, Division

* 📐 Scientific Functions

  * Trigonometric: sin, cos, tan
  * Logarithmic: log, ln
  * Exponential calculations
  * Power & square root

* 🧠 Advanced Capabilities

  * Factorial
  * Modulus
  * Parentheses handling
  * Operator precedence

* 💻 User-Friendly Interface

  * Clean and simple UI
  * Easy input handling
  * Fast and responsive

## ⚙️ How to Run

### ▶️ For Java Version

1. Compile the code:

   ```
   javac Main.java
   ```
2. Run the program:

   ```
   java Main
   ```

### ▶️ For C Version

1. Compile:

   ```
   gcc calculator.c -o calc
   ```
2. Run:

   ```
   ./calc
   ```
